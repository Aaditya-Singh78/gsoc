1. Clone the project
2. Install dependencies
    - `yarn install`
3. Start development server
    - `yarn start`
4. Setup CORS in FOSSology
    - Open FOSSology and goto Admin > Customize.
    - Lookout for "Allowed origins for REST API"
    - Set the value to "http://localhost:3000"
